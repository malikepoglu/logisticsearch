# Parse Core SQL Surface

## Overview

This directory is the canonical repository surface for the parse-layer SQL of LogisticSearch. Its purpose is to bring the live Pi51 parse schema into the GitHub-centered repository as imported evidence, and then prepare it for later disciplined decomposition and controlled evolution.

## Genel Bakış

Bu dizin, LogisticSearch’in parse katmanı SQL yüzeyi için kanonik repository alanıdır. Amacı, canlı Pi51 parse şemasını GitHub merkezli repository içine ithal edilmiş kanıt olarak almak ve daha sonra disiplinli parçalama ile kontrollü evrim için hazırlamaktır.

## Current status

At this stage, the surface is expected to begin with a live imported snapshot from Pi51, plus inventory and checksum material.

## Mevcut durum

Bu aşamada yüzeyin, Pi51’den ithal edilmiş canlı snapshot ile birlikte inventory ve checksum materyaliyle başlaması beklenir.
